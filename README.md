# NeuroGate API 1.1

Небольшой Windows-оверлей для лимитов NeuroGate API.

English note: this is a compact Windows overlay for NeuroGate API usage limits.
The main documentation is in Russian because the primary users are Russian-speaking.

Оверлей читает страницу `https://portal.neurogate.space/client/usage` через
локальный профиль Chrome и показывает маленькое окно поверх рабочего стола:
название тарифа, остатки кредитов, время до сброса и интервал обновления.
После входа видимое окно Chrome обычно скрывается, а данные продолжают
обновляться из той же локальной браузерной сессии.

## Что показывает оверлей

Текущая страница лимитов NeuroGate отдаёт меньше данных, чем старая версия
интерфейса. Поэтому оверлей намеренно показывает только то, что реально видно
на странице сейчас:

- название тарифа;
- сколько времени осталось до окончания тарифа;
- остаток кредитов за 5 часов;
- остаток кредитов за 7 дней;
- полоски прогресса, совпадающие с полосками на сайте NeuroGate;
- время до сброса каждого лимита;
- время последнего обновления;
- выбранный интервал обновления.

Старые поля вроде общего количества токенов, кеша и пары `использовано / всего`
не показываются, если новая страница их больше не отдаёт в видимом интерфейсе.
Оверлей не показывает старые сохранённые значения вместо свежих данных: если
сайт требует вход или не отдаёт лимиты, в окне будет показано это состояние.

## Приватность

Проект работает локально на твоём компьютере.

- Оверлей не просит пароль NeuroGate в своём интерфейсе.
- Оверлей не собирает API-ключи.
- Оверлей не отправляет лимиты, cookies или данные аккаунта в этот проект,
  на отдельный сервер или в аналитику.
- Он читает только текст и элементы, уже видимые в твоей браузерной сессии.
- Cookies остаются на компьютере в локальном профиле Playwright/Chrome.
- После успешного входа видимое окно Chrome скрывается, а последующие чтения
  идут из той же локальной сессии.
- В меню по правой кнопке есть пункт `Не закрывать ЛК`. Если включить его,
  личный кабинет NeuroGate останется открытым отдельным окном Chrome. Если
  выключить, окно снова будет скрыто.
- Старые снимки лимитов не используются как запасные данные.

Локальный профиль браузера по умолчанию:

```text
%USERPROFILE%\.neurogate-usage-overlay\browser-profile
```

Эту папку нельзя публиковать или передавать другим людям: в ней может быть
твоя локальная браузерная сессия.

## Требования

- Windows 10/11
- Python 3.10+
- Google Chrome
- доступ к сайту NeuroGate

## Установка

Через GitHub:

```powershell
git clone https://github.com/RyandavisProject/neurogate-overlay.git
cd neurogate-overlay
powershell -ExecutionPolicy Bypass -File .\scripts\install.ps1
```

Или двойным кликом:

```text
scripts\install.bat
```

Установщик создаёт:

- локальное Python-окружение в `.venv/`;
- установку пакета в режиме разработки;
- ярлык на рабочем столе с названием `NeuroGate API`.

## Запуск

Через PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\run-overlay.ps1
```

Или двойным кликом по ярлыку:

```text
NeuroGate API
```

Первый запуск:

1. Оверлей пытается прочитать страницу лимитов в скрытом режиме.
2. Если нужен вход, откроется отдельное окно Chrome с локальным профилем.
3. Войди на сайте NeuroGate обычным способом.
4. После первого успешного чтения видимое окно Chrome будет скрыто.
5. Дальше обновления будут идти из той же локальной сессии.
6. Пока нужен вход, оверлей проверяет состояние каждые несколько секунд и
   быстро подхватывает успешную авторизацию.
7. После появления свежих данных автообновление идёт не чаще одного раза в
   минуту, если вручную не выбран другой интервал.

Если нужно держать личный кабинет открытым, нажми правой кнопкой по оверлею и
включи `Не закрывать ЛК`. Если выключить этот пункт, окно Chrome снова будет
скрыто. Настройка действует только во время текущего запуска: при следующем
обычном запуске приложение снова стартует в скрытом режиме, если не использовать
параметр `--show-browser`.

## Управление

- Оверлей можно перетаскивать за любую видимую область.
- Последняя позиция окна сохраняется локально и восстанавливается при запуске.
- Левый клик по кнопке интервала, например `1м`, переключает интервалы.
- Правый клик открывает компактное меню.
- Пункт `Не закрывать ЛК` держит страницу NeuroGate открытой отдельным окном.
- `Esc` закрывает оверлей.
- `Ctrl+R` запускает ручное обновление с учётом минимальной паузы в 1 минуту.

## Разработка

Проверка проекта:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\check.ps1
```

Тесты сейчас покрывают оба известных формата страницы:

- старую страницу NeuroGate с `24 часа / 7 дней` и парой лимитов тарифа;
- текущую страницу NeuroGate с `5 часов / 7 дней`, остатками кредитов и
  полосками прогресса.

## Ограничения

- Парсер читает видимый текст и элементы страницы. Если NeuroGate изменит
  подписи или структуру интерфейса, проекту может понадобиться небольшое
  обновление.
- Приложение ориентировано на Windows, потому что использует поведение Tkinter
  для десктопного окна и Windows-скрипты для ярлыка.
- Оверлей не обходит вход и не продлевает сессию сайта. Если NeuroGate разлогинит
  пользователя, откроется видимое окно Chrome для повторного входа. После
  успешного чтения окно снова скрывается.
- Оверлей не показывает старые fallback-значения. Если портал требует вход или
  не отдаёт лимиты, виджет показывает это состояние вместо устаревших цифр.
